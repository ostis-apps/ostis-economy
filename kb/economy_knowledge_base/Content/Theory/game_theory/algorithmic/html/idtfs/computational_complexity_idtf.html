Вычислительная сложность - это понятие, которое описывает количество вычислений, необходимых для решения игровых задач. Она определяется как объем вычислений, необходимых для определения оптимальных стратегий для каждого игрока, при условии, что каждый игрок выбирает свою оптимальную стратегию, учитывая стратегии других игроков.
